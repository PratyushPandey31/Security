01 — Executive Summary

Project Overview

CloudShield v4 is a production-grade, zero-trust cloud security virtualization suite built for cybersecurity education, demonstration, and research. It simulates a real-world enterprise Security Operations Center (SOC) with live threat detection, active deception, cryptographic data protection, and automated vulnerability scanning.

The platform is fully deployed and accessible publicly. The SIEM dashboard on Vercel connects to a local microservice backend through an auto-established HTTPS tunnel, enabling real-time telemetry from any network without browser security restrictions.

02 — System Architecture

Microservices Architecture

CloudShield follows a microservices pattern where each security component runs as an independent service. Traffic flows through the WAF Proxy Gateway, which routes requests to either the legitimate web application or the deception honeypot depending on behavior patterns.

03 — Core Features

Platform Capabilities

🛡️
WAF Proxy Gateway — RegEx-based rule engine blocks SQLi, XSS, path traversal, brute-force patterns in real-time
🍯
Active Deception Honeypot — Transparent redirect to mock terminal sandbox captures attacker command executions
🔐
Cryptographic Vault — RC4 stream cipher encrypts all uploaded files at-rest inside SQLite with session-key derivation
📡
SSE Telemetry Stream — Real-time Server-Sent Events feed powers live dashboard updates without polling
🗺️
Live Threat Map — SVG world map with dynamic Bezier curved attack vector animations per incoming alert
📊
Analytics Suite — 6 Chart.js visualizations: area charts, doughnuts, horizontal bars with gradient fills
⚡
Vulnerability Auditor — Multi-stage pentest scanner with conic radar UI, grades A–F, CSV export
🔌
Auto HTTPS Tunnel — SSH tunnel to localhost.run starts in background thread, URL auto-pushed to GitHub
👤
Zero-Trust Auth — Glassmorphic login with rotating SVG lock, local storage session vault, secure logout
🚫
IP Ban System — Automatic firewall bans from honeypot triggers, manual unban via SOC dashboard
🔑
MFA Simulation — 6-digit time-based verification flow matching standard authentication workflows
🐳
Docker Support — Full docker-compose.yml for one-command container cluster deployment

🏁 Conclusion

CloudShield v4 successfully demonstrates a comprehensive enterprise-grade cybersecurity stack. The platform integrates six distinct security disciplines — network proxying, intrusion deception, cryptographic protection, real-time monitoring, vulnerability assessment, and security analytics — into a cohesive, production-deployable microservices architecture.

The SIEM dashboard is publicly accessible and provides a visually stunning, data-rich interface that rivals commercial security platforms. The system is fully extensible for additional WAF rules, detection algorithms, or new service integrations.
